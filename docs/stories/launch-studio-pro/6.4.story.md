## Story

**As a** usuário premium do Launch Studio PRO,
**I want** baixar um ZIP com todos os assets do dossiê (e-mails em MD/HTML, scripts PLC, página de vendas e checklist),
**so that** eu tenha todos os materiais prontos para uso em um único arquivo (PRD §2).

## Acceptance Criteria

1. API route `POST /api/launch-studio-pro/projects/[id]/export/zip` gera um ZIP contendo todos os artefatos do dossiê.
2. ZIP inclui: e-mails de todas as fases (`.md` e `.html`), scripts de vídeo PLC 1/2/3 (`.md`), página de vendas (`.html`), checklist de execução (`.md`).
3. Estrutura de pastas dentro do ZIP: `emails/`, `scripts/`, `sales-page/`, `checklist/`.
4. ZIP salvo no storage via módulo de 6.2 e URL de download retornada.
5. Botão "Baixar Pack Completo" na tela de preview/dashboard com loading state.
6. `npm run lint` e `npm run typecheck` passam.
