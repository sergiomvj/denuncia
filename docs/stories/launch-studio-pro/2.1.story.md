## Story

**As a** usuário premium do Launch Studio PRO,
**I want** ver meu dashboard "Meus Lançamentos" com a lista dos meus projetos e um botão "+ Novo Lançamento",
**so that** eu possa gerenciar meus projetos de lançamento e iniciar novos de forma rápida (PRD §3.3, Fluxo passo 3).

## Acceptance Criteria

1. Rota `/app/launch-studio-pro` (ou equivalente) renderiza o dashboard "Meus Lançamentos" para usuários premium autenticados.
2. Dashboard exibe lista de projetos do usuário (da tabela `launch_projects`) filtrados por `user_id` atual e `deleted_at IS NULL`.
3. Cada item da lista exibe: título do projeto, status atual, data de criação.
4. Lista ordenada por `updated_at DESC` (mais recentes primeiro).
5. Estado vazio: quando não há projetos, exibe mensagem amigável e CTA "+ Novo Lançamento" destacado.
6. Estado de loading: exibe skeleton/loader enquanto projetos são carregados.
7. CTA "+ Novo Lançamento" presente e visível (mobile-first) — navega para o fluxo de criação de projeto (story 2.2/2.3).
8. Filtro visual de status: usuário pode filtrar projetos por status (draft, interviewing, ready, launched, archived) — pelo menos como botões/tabs.
9. Layout mobile-first com Design System v2 (Bricolage Grotesque + Inter, tema escuro).
10. `npm run lint` e `npm run typecheck` passam.
