## Story

**As a** usuário não-premium que tenta acessar o Launch Studio PRO,
**I want** ser redirecionado para uma tela de upgrade clara com CTA para assinar,
**so that** eu entenda por que não tenho acesso e possa facilmente fazer upgrade para premium (PRD §0, §12 UX).

## Acceptance Criteria

1. Página `/upgrade` (ou `/upgrade?app=launch-studio-pro`) existe e renderiza corretamente.
2. Mensagem clara explica que o Launch Studio PRO é um recurso exclusivo Premium.
3. CTA principal ("Assinar Premium" ou equivalente) leva ao fluxo de checkout Stripe existente (REUSE `app/api/stripe/*`).
4. Banner/componente "Recurso Premium" reutilizável pode ser usado em outras rotas do app (fallback UI — PRD §0).
5. Layout usa Design System v2: fonte Bricolage Grotesque para títulos, Inter para corpo, tema escuro com gradiente de marca (PRD §12 UX/UI).
6. Tela é mobile-first e responsiva (PRD §12 UX/UI).
7. `npm run lint` e `npm run typecheck` passam.
8. Usuário premium que acessa `/upgrade` diretamente é redirecionado para o dashboard do app (não fica preso na tela de upgrade).
