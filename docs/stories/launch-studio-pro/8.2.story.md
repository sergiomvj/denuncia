## Story

**As a** desenvolvedor do Launch Studio PRO,
**I want** garantir que os critérios técnicos do PRD §14 são atendidos com testes unitários/integração das fases críticas e lint/typecheck passando,
**so that** o app tenha qualidade verificável antes do aceite formal (PRD §14).

## Acceptance Criteria

1. `npm run lint` passa sem erros em todos os arquivos do Launch Studio PRO.
2. `npm run typecheck` passa sem erros.
3. Testes unitários do gate premium (middleware `premiumGate`) cobrem: usuário premium, não-premium, não autenticado.
4. Testes de isolamento RLS: usuário A não acessa dados de usuário B.
5. Testes do orquestrador multi-LLM (com mocks): roteamento por fase, fallback, cost tracker.
6. Testes de exportação: geração de Markdown, CSV, ICS com fixtures de dossiê completo.
7. Cobertura de teste mínima nos módulos críticos: `lib/launch-studio-pro/` com pelo menos 1 teste por módulo.
8. Testes de esquema Zod: dossiê válido e inválido.
