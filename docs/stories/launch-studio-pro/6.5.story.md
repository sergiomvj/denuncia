## Story

**As a** usuário premium do Launch Studio PRO,
**I want** exportar o cronograma do lançamento como ICS (importável no Google Calendar/Apple Calendar) e CSV,
**so that** eu possa planejar a execução do lançamento diretamente no meu calendário pessoal (PRD §2, §7).

## Acceptance Criteria

1. API route `POST /api/launch-studio-pro/projects/[id]/export/ics` gera arquivo ICS válido com os eventos do calendário do dossiê.
2. API route `POST /api/launch-studio-pro/projects/[id]/export/csv` gera arquivo CSV com as colunas: Dia, Fase, Ação, Asset.
3. ICS gerado é importável no Google Calendar (validado com formato RFC 5545).
4. Google Calendar API é opcional (sem dependência obrigatória — fallback é o arquivo ICS estático) — PRD §7.
5. CSV inclui uma linha por entrada do `payload.calendar` do dossiê.
6. Artefatos salvos no storage via módulo de 6.2.
7. `npm run lint` e `npm run typecheck` passam.
