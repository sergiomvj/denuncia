## Story

**As a** desenvolvedor do Launch Studio PRO,
**I want** implementar rate limiting de 50 gerações/dia por usuário e CAPTCHA em padrão suspeito,
**so that** o sistema seja protegido contra abuso e uso excessivo que eleve os custos (PRD §11.6, §11.1).

## Acceptance Criteria

1. Rate limit de 50 gerações/dia por usuário implementado server-side na rota de geração (PRD §11.6).
2. Se usuário exceder 50 gerações/dia: retornar 429 com mensagem "Limite diário de gerações atingido. Tente novamente amanhã."
3. Rate limit de 5 gerações simultâneas por usuário (já em 7.2 — esta story trata do limite diário) (PRD §11.6).
4. Padrão suspeito detectado: se usuário faz 10+ tentativas de acesso negado em `premium_access_logs` em menos de 5 minutos, acionar indicação de CAPTCHA (PRD §11.1).
5. CAPTCHA implementado como indicação server-side (ex: response header `X-Require-Captcha: true`) — frontend exibe CAPTCHA quando flag presente. CAPTCHA real pode ser hCaptcha ou equivalente.
6. Rate limit implementado via Redis/Upstash ou contador em banco (com TTL de 24h).
7. `npm run lint` e `npm run typecheck` passam.

## Dev Notes

### Opção recomendada para rate limit (Redis/Upstash)
Upstash Redis é serverless-friendly e compatível com Vercel. Verificar se já está no projeto.

```ts
// lib/launch-studio-pro/rate-limit.ts
import { Redis } from '@upstash/redis';
const redis = Redis.fromEnv();

export async function checkDailyGenerationLimit(userId: string): Promise<boolean> {
  const key = `ratelimit:gen:${userId}:${new Date().toISOString().slice(0, 10)}`;
  const count = await redis.incr(key);
  if (count === 1) await redis.expire(key, 86400); // 24h TTL
  return count <= 50;
}
```

### Alternativa sem Redis
Se Redis não estiver disponível, usar tabela `user_daily_rate_limits(user_id, date, count)` com upsert — mais lento mas funcional.

### Dependência
Depende de 7.2 (limite de concorrência), 1.2 (tabela de logs de acesso para detecção de padrão suspeito).

### Testing

- 10 acessos negados em 5 min: flag CAPTCHA ativada
- 9 acessos negados em 5 min: sem flag CAPTCHA
