## Story

**As a** usuário premium do Launch Studio PRO,
**I want** regenerar uma fase específica do dossiê, favoritar, copiar e baixar artefatos individualmente,
**so that** eu possa refinar partes do plano sem regerar tudo, respeitando os limites do meu plano (PRD §2 ações, §3.10).

## Acceptance Criteria

1. API route `POST /api/launch-studio-pro/projects/[id]/dossier/[dossierId]/regenerate/[phaseSlug]` regenera uma fase específica (ex: `plc1`, `copywriter`, `avatar`) sem alterar as outras.
2. Regeneração verifica limite de regenerações do plano (`user_premium_limits.max_regenerations_per_month`) antes de executar (PRD §10).
3. Se limite atingido: retorna 429 com modal de upgrade (mesmo padrão de 2.4).
4. Após regeneração: cria novo registro em `launch_dossiers` com versão incrementada (versionamento de 5.7).
5. Ação "Favoritar": botão de favorito no preview que salva `launch_projects.is_favorited = true` (ou campo equivalente).
6. Ação "Copiar": copiar texto de uma seção para o clipboard (client-side, sem API).
7. Ação "Baixar": link para download do artefato correspondente (PDF, MD, ZIP — reusar routes de 6.3/6.4).
8. `npm run lint` e `npm run typecheck` passam.
