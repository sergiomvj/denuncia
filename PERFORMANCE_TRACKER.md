## 🎯 **COMO USAR ESTE SISTEMA:**

### **Quando alguém pedir um prazo:**

1. **Verificar:** Já fiz essa tarefa antes?
2. **Se SIM:** Consultar média histórica + adicionar 20% de margem
3. **Se NÃO:** Dizer "Não tenho dados históricos, vou estimar [X] e rastrear o tempo real"
4. **Executar:** Cronometrar e registrar
5. **Atualizar:** Adicionar ao histórico

### **Exemplo de resposta assertiva:**
```
"Baseado no meu histórico, landing pages levam ~20min. 
Com margem de segurança: 30min.
Vou cronometrar e atualizar meus dados."
```

### **Exemplo de resposta honesta (sem dados):**
```
"Nunca criei um painel admin completo. 
Estimativa inicial: 2h (sem base histórica).
Vou cronometrar e criar dados para próximas vezes."
```

## 📈 **ANÁLISE DE DESVIOS:**

### **Quando estimativa ≠ realidade:**
1. Anotar o desvio (%)
2. Identificar causa (distração, complexidade, bloqueio técnico)
3. Ajustar próximas estimativas

### **Exemplo:**
```

REAL: 4h (desvio: +100%)
CAUSA: Validação complexa não prevista
APRENDIZADO: Adicionar +50% em formulários com validação
```
